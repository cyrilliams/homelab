## Goals

In this repo, I'll document my experience setting up my server(s) using ProxMox, Windows and Ubuntu servers, docker containers, a NAS setup, PfSense and more.

Most of this will be for home use and function but I would also like to increase my knowledge of networking as well as simululate a real world company/organization on-prem, then hybrid evironment.

I do have prior experience setting up most to all of these as I have setup servers before but I will be rebuilding everything from the ground up.

## Projects

A couple projects I would like to setup will be:

- Setup ProxMox (or VM-Ware ESXi, or both)
- Setup Windows Server, Domain Controller, Active Directory, add my personal devices on to a domain
- Sync my on prem DC/AD to Entra/Azure using Entra Connect (if my licence is still available)
- Setup double NAT to use a different network than my regular
- Setup PfSense as my Router and Firewall
- Setup docker containers to use for my home environment
- Setup my Domain/DNS to point to specific containers
- Allows internet traffic to come to docker containers or services on my server using port forwarding.
- Configure subnets
- Configure a VLAN used for a home guest network
- Configure some sort of security controls
- Implement network monitoring
- Setup a SAN and connect it to hypervisor to be used for storage.
